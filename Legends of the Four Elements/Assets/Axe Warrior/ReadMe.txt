
Armor Set Naming Conventions
The armor sets are prefixed for easy identification:

A1_ → Armor Set 1

A2_ → Armor Set 2

A3_ → Armor Set 3

All armor pieces are grouped under empty GameObjects:

Helmet_Equipments → Contains all helmets from A1, A2, A3

Body_Equipments → Contains Body armor

Hand_Equipments → Contains all hand armor

Leg_Equipments → Contains all leg armor
